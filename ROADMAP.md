# 🗺 功能路线图

## ✅ 已实现（MVP v1.0）

### 核心功能
- [x] **🧭 模块1：价值观（Why）** — 5步找到人生指南针
  - 5个引导问答，AI 提取关键词
  - 关键词归纳为核心价值观
  - 真假价值观辨别（可控性检测）
  - 优先级排序 + 可视化金字塔
  - 确定工作目的
- [x] **⚡ 模块2：才能（How）** — 发现无意识天赋
  - 5个引导问答，AI 提取才能
  - 缺点转优点转换器
  - 成功经历8角度深挖
  - 《自己的使用说明书》表格（◎〇△分级）
- [x] **❤️ 模块3：热情（What）** — 探索兴趣领域
  - 5个引导问答，AI 提取兴趣
  - 合理性陷阱检测
- [x] **🔗 模块4：组合筛选** — What × How × Why
  - 自由组合生成器
  - 工作目的筛选
  - AI 智能推荐组合
  - 最小验证计划制定

### 系统功能
- [x] **🤖 AI 智能分析** — 接入 MiniMax API，自然语言→关键词提取
- [x] **👥 多用户支持** — 新建/切换/删除用户，数据独立存储
- [x] **📊 个人仪表盘** — 完整自我认知报告总览
- [x] **📱 移动端适配** — 响应式布局，触摸优化
- [x] **💾 本地持久化** — 浏览器 localStorage 存储，刷新不丢数据
- [x] **🔄 旧数据迁移** — 自动检测并迁移旧版存储格式

### v1.1 新增
- [x] **📄 PDF / Markdown 报告导出** — 完整分析结果导出为可分享文档
- [x] **⚙️ 自定义 AI 设置** — 用户可选择 DeepSeek/MiniMax/通义千问/腾讯混元/硅基流动，使用自己的 API Key
- [x] **🔐 安全护栏** — pre-commit 敏感信息检测 + GitHub Actions 自动扫描
- [x] **🔒 隐私说明** — 首页/仪表盘/README 明确标注数据仅存本地
